# Introduzione

Questo sito contiene una raccolta di appunti, esercizi svolti ed approfondimenti per la parte di laboratorio del corso "Reti di calcolatori", tenuta dal professor Christian Quadri all'Università degli studi di Milano.

Non è da ritenersi sostitutivo del materiale di studio di teoria, ma un solo aiuto nella comprensione e nello svolgimento degli esercizi di Packet Tracer e Wireshark.

Chiunque può contribuire proponendo merge request al repo GitHub associato.
